【学术讲座】师江帆做关于“Integration Testing of Software Product Lines Using Compositional Symbolic Execution”主题讲座

发布时间：2017年11月30日 10:00    作者：    来源：     点击率：次

     2017年11月24号，我院邀请到师江帆博士在主楼11210教室做关于“Integration Testing of Software Product Lines Using Compositional Symbolic Execution”的学术讲座。

 

师江帆现担任龙测科技创始人兼 CEO。在 1998，他在华北工学院计算机应用获得工学学士，在 2005 获得內布拉斯加大学奥马哈分校计算机科学系理学硕士，并在 2012 获得內布拉斯加大学林肯分校计算机科学与工程系理学博士。

他分别在学术界和工业界的软件测试领域工作过。基于他发表的论文，谷歌学者显示有分布在各大著名大学或研究机构超过 600 多引用，比如卡内基梅隆大学，斯坦福大学，伦敦大学学院以及微软研究院。从2012 到 2014，他曾在微软 Dynamics 组做软件测试工程师和 Bing 组做软件开发工程师, 之后在 Expedia 任资深软件测试工程师职位。他同时因在软件测试领域的国际声望而以杰出人才方式获得美国绿卡。

具体工作经历如下：

2014/11-2016/4 Expedia 北京分公司 资深软件测试工程师

2012/2-2014/10 美国微软 软件测试/开发工程师

2010/5-2010/8 美国 IBM 实习生

学术经历：

2005/8- 2012/5 内布拉斯加大学-林肯分校 计算机科学 博士

2003/1-2005/8 内布拉斯加大学-奥马哈分校 计算机科学 硕士

1994/09-1998/07 华北工学院 计算机科学与技术，学士

在美国攻读计算机硕/博期间，师江帆共发表高水平论文 7 篇。
